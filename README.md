## Использование

Установите telebot с помощью команды:
```sh
$ pip install pyTelegramBotAPI
```

И укажите токен вашего бота:
```typescript
TOKEN = 'Ваш_Токен' 
bot = telebot.TeleBot(TOKEN)
```
